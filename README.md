# Quiz Blitz Arena

Quiz Blitz Arena is a real-time multiplayer quiz application.

## Tech Stack

- **Frontend:** React (Vite), TypeScript, Tailwind CSS, Shadcn UI, Framer Motion, React Query, Socket.IO Client.
- **Backend:** Node.js, Socket.IO.
- **Testing:** Vitest, React Testing Library.

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18+ recommended)
- npm

### Installation

1. Clone the repository and install dependencies:
   ```bash
   npm install
   ```

2. Start the development server (Frontend):
   ```bash
   npm run dev
   ```

3. To build the frontend and run the production server:
   ```bash
   npm run start:server
   ```

## Available Scripts

- `npm run dev`: Starts the Vite development server.
- `npm run build`: Builds the app for production.
- `npm run start:server`: Builds the app and starts the backend server.
- `npm run pm2:deploy`: Deploys the application using PM2.
- `npm run test`: Runs the Vitest test suite.
- `npm run lint`: Runs ESLint to check for code quality.
